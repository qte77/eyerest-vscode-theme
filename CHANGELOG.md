# Changelog

## [0.1.0] - 2026-03-28

### Added
- EyeRest Dark theme — warm brown backgrounds with muted syntax colors
- EyeRest Light theme — warm cream/ivory backgrounds for well-lit environments
- EyeRest Green Dark theme — deep forest green with all-green palette, green text
- EyeRest Green Light theme — soft mint-green backgrounds with deep green text
- EyeRest Amber Dark theme — zero-blue palette for 100% blue-blocking goggle users
- EyeRest Amber Light theme — warm parchment with zero-blue syntax colors
- Semantic highlighting support for modern language servers
- Full 16-color ANSI terminal palette for all themes
- Bracket pair colorization with 6 distinct colors
